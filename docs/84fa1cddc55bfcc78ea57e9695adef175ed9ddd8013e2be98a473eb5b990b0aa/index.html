Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 85 brownfield sites. 13 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 94.13 hectares of brownfield land
  • 3503 potential dwellings that could be built
  • 0 sites have permission in principle
