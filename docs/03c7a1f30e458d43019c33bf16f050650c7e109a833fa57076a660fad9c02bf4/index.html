Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 48 brownfield sites. 0 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 57.25 hectares of brownfield land
  • 2483 potential dwellings that could be built
  • 0 sites have permission in principle
