Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 18 brownfield sites. 0 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 17.94 hectares of brownfield land
  • between 638 and 678 potential dwellings that could be built
  • 4 sites have permission in principle
