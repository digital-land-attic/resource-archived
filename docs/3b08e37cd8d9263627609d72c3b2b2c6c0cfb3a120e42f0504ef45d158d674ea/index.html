Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 37 brownfield sites. 0 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 44.66 hectares of brownfield land
  • between 836 and 1037 potential dwellings that could be built
  • 1 site has permission in principle
