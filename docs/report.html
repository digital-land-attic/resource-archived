Brownfield collector report

Last updated 27 November 2020

Summary

Since our collector start running regularly it has collected brownfield land data from 345 Organisations.

We have found brownfield data behind 868 Urls.

And there have been 984 different resources, where a resource is either a new or updated brownfield site register.
