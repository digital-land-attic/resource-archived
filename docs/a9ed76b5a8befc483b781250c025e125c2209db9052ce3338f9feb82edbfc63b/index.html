Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 29 brownfield sites. 1 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 12.78 hectares of brownfield land
  • between 448 and 569 potential dwellings that could be built
  • 0 sites have permission in principle
