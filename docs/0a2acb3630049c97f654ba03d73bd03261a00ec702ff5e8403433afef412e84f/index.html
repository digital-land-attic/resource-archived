The following information has been extracted from the register.

The register references 40 organisations.

It reports 43 brownfield sites. 0 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 22.77 hectares of brownfield land
  • 814 potential dwellings that could be built
  • 0 sites have permission in principle
