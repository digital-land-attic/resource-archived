About the data

This register has reported a total of 23 brownfield sites.

8 of these have an end date and are no longer considered brownfield land.

Excluding sites with an end date, there are 10.56 hectares of land.

This land has the potential to provide between 349 and 541 dwellings.

0 of these sites have permission in principle.
