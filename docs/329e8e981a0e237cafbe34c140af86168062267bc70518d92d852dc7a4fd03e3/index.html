Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 102 brownfield sites. 34 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 167.66 hectares of brownfield land
  • between 1558 and 2407 potential dwellings that could be built
  • 0 sites have permission in principle
