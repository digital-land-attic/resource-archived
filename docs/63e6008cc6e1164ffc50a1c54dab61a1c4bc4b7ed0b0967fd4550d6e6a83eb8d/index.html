Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 75 brownfield sites. 7 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 92.71 hectares of brownfield land
  • between 3411 and 3392 potential dwellings that could be built
  • 0 sites have permission in principle
