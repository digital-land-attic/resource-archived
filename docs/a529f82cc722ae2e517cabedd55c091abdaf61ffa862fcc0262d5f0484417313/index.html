Facts from the data

The following information has been extracted from the register.

The register references 1 organisation.

It reports 32 brownfield sites. 0 of these have an end date and are no longer considered brownfield land.

Not including sites with an end date, there are:

  • 72.00 hectares of brownfield land
  • 1808 potential dwellings that could be built
  • 0 sites have permission in principle
